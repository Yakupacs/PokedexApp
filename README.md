## Description
<p>You can list Pokémon and view them in detail using the Pokémon application.</p>

## Technologies and Libraries
- Protocol Oriented Programming
- URLSession
- Programmatic UI
- MVVM
- Delegate Pattern
- Singleton Pattern
- Unit Tests
- Kingfisher
